# How I Build

An interactive visualization of my AI-enabled product development workflow — from discovery through prototyping and validation.

Built with React + Vite. Zero dependencies beyond React itself.

## Run locally

```bash
npm install
npm run dev
```

## Reskin

All brand colors are defined as constants at the top of `src/components/WorkflowDiagram.jsx`. Swap ~6 lines to match any company's style guide.
